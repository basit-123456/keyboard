# Modern Virtual Keyboard

A beautiful, fully functional virtual keyboard built with HTML, CSS, and JavaScript featuring modern design and smooth animations.

## Features

- **Modern Glass-morphism Design** - Beautiful translucent interface with blur effects
- **Fully Functional** - All standard keyboard keys including modifiers
- **Physical Keyboard Support** - Works with both mouse clicks and physical keyboard input
- **Visual Feedback** - Keys highlight and animate when pressed
- **Responsive Design** - Adapts to different screen sizes and devices
- **Special Keys Support** - Shift, Caps Lock, Tab, Enter, Backspace, and more
- **Real-time Display** - Live text output in textarea

## Files

- `index.html` - Main HTML structure
- `style.css` - Modern CSS styling with animations
- `script.js` - JavaScript functionality and event handling

## Usage

1. Open `index.html` in your web browser
2. Start typing using either:
   - Click keys with your mouse
   - Use your physical keyboard
3. All standard keyboard functions work:
   - Shift for capitals and symbols
   - Caps Lock toggle
   - Backspace to delete
   - Enter for new lines
   - Tab for indentation

## Browser Compatibility

- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## Customization

Easily customize colors, animations, and layout by modifying the CSS variables and classes in `style.css`.
